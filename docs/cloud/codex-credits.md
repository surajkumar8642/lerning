# Codex Credits by Plan

Last checked: March 22, 2026

## Free

- Codex is included with ChatGPT Free for a limited time.
- OpenAI's current help article describes this as included access, not a separate credit balance.
- The same article says other paid plans get higher rate limits.

Source:
- https://help.openai.com/en/articles/11369540-using-codex-with-your-chatgpt-plan

## Pro

- Pro includes Codex access.
- OpenAI says Plus, Pro, Business, and Enterprise/Edu plans enjoy 2x rate limits compared with the limited-time Free/Go access.
- Flexible usage credits for paid individual plans are documented separately by OpenAI. If you need exact Pro credit pricing, check the current ChatGPT flexible usage article in the Help Center because these values can change.

Source:
- https://help.openai.com/en/articles/11369540-using-codex-with-your-chatgpt-plan
- https://help.openai.com/en/articles/11481834

## Enterprise

- Enterprise includes Codex access.
- If flexible pricing is enabled, usage is billed through the workspace credit/rate-card system instead of a simple personal credit bucket.
- OpenAI's current rate card shows ChatGPT workspace credit pricing, and the article links to a separate Codex rate card for Codex-specific charges.
- OpenAI release notes also say Enterprise/Edu users without flexible pricing currently get 2x Codex rate limits for a limited time.

Source:
- https://help.openai.com/en/articles/11481834
- https://help.openai.com/en/articles/10128477-chatgpt-enterprise-edu-release-notes

## Short Summary

- Free: limited-time included Codex access, no separate published personal credit bucket in the plan overview.
- Pro: included Codex access with higher limits; credit-based flexible usage exists for paid individual plans.
- Enterprise: included Codex access; credits/rates depend on workspace billing and flexible pricing settings.

## Notes

- OpenAI updates these plan details frequently.
- If you want, this file can be expanded with exact current rate-card numbers once you confirm whether you want ChatGPT credits, Codex rate-card pricing, or both.
