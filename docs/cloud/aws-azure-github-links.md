# AWS and Azure GitHub Links

Last checked: March 23, 2026

Purpose:

- official GitHub links
- sample code repositories
- broad entry points instead of cloning everything

## Important Note

There is not one single repo containing all AWS samples or all Azure samples.

The practical approach is:

- use official GitHub organizations
- use official umbrella sample repos
- then open specific service repos from there

## AWS Official GitHub Entry Points

- AWS Samples organization: https://github.com/aws-samples

## How to Use These Links

If you want broad AWS learning:

1. open `awsdocs/aws-doc-sdk-examples`
2. open `aws-samples/serverless-samples`
3. then search inside by service name like:
   - lambda
   - ecs
   - ecr
   - eventbridge
   - step functions
   - cognito
   - iam
   - s3

If you want broad Azure learning:

1. open `Azure-Samples`
2. search by service name like:
   - app service
   - functions
   - aks
   - openai
   - ai search
   - devops
   - storage
   - key vault
